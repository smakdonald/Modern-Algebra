    Section 2.1 Linear Independence

    Subsection Generated Modules

    \(\defn\) ‚Äì Linear Combination.

    Let \(M\) be an \(R\)-module and \(m_1, \dots, m_n \in M\text{.}\) An \(R\)-\(\defnn{linear combination}\) of \(m_1, \dots, m_n\) is an element of \(M\) of the form \(r_1m_1 + \cdots + r_nm_n\) for some \(n \geq 0\) and \(r_1,\dots,r_n \in R\text{.}\) (If \(n = 0\text{,}\) this gives the empty sum which is interpreted to give \(0_M\text{.}\))
    \(\rem\).
    If \(M\) is an \(F\)-vector space, one usually uses the term ‚Äúspanned by‚Äô‚Äô instead of the term‚Äùgenerated by‚Äô‚Äô.
    \(\lem\) ‚Äì \(R\cdot A\) Smallest Submodule of \(M\) Containing \(A\).
    For any subset \(A\) of a \(R\)-module \(M\text{,}\) the subset \(R \cdot A\) is indeed a submodule of \(M\text{,}\) and it is the smallest submodule of \(M\) that contains \(A\) as a subset. In fact, we have
    \begin{equation*} R \cdot A = \bigcap\limits_{A\subseteq N, N \text{ submodule of }M} N. \end{equation*}
    \(\defn\) ‚Äì Finitely Generated (Module).
    A module \(M\) is \(\defnn{finitely generated}\) if there exists some finite subset \(A\) of \(M\) such that \(M = R \cdot A\text{.}\)
    \(\rem\).
    For any \(R\)-module \(M\text{,}\) we have \(R \cdot \emptyset = \{0_M\}\text{.}\) This is because the empty sum is interpreted as giving \(0_M\text{.}\)
    \(\defn\) ‚Äì Cyclic Module.
    If \(M = R \cdot a\)[^1] for some single element \(a \in M\text{,}\) we say that \(M\) is \(\defnn{cyclic}\).
    \(\exe\) ‚Äì Cyclic \(\Z\)-module.
    If \(R = \Z\text{,}\) then (recalling that a \(\Z\)-module is the same thing as an abelian group) we see that \(M\) is a cyclic \(\Z\)-module if and only if \(M\) is a cyclic group.
    Problem ‚Äì.
    Show that the left \(R\)-module \(M\) is cyclic if and only if there exists a left ideal \(I\) of \(R\) such that \(M \cong R/I\text{.}\)
    Problem ‚Äì.
    Let \(R\) be a commutative a ring. Let \(M\) be an \(R\)-module, \(I\) an ideal in \(R\text{,}\) and let
    \begin{equation*} N = \{ m \in M \mid am = 0 \textrm{ for all } a \in I \}, \end{equation*}
    which is an \(R\)-submodule of \(M\) by Problem 1. Show that
    \begin{equation*} \Hom_R(R/I,M) \cong N. \end{equation*}
    Problem 5 ‚Äì Annihilators and Generated Ideals.
    Let \(R\) be a (not-necessarily commutative) ring let \(M\) be a left \(R\)-module. The annihilator of \(M\) in \(R\) is defined to be
    \begin{equation*} \ann_R(M) = \{ r \in R \mid rm = 0 \text{ for all $m \in M$}\}. \end{equation*}
    (a) Prove that \(\ann_R(M)\) is a \(2\)-sided ideal of \(R\text{.}\) (b) Suppose \(M\) is an abelian group (i.e., a \(\Z\)-module) such that \(|M| = 400\) and \(\ann_\Z(M)\) is the ideal generated by \(20\text{.}\) How many possibilities, up to isomorphism, are there for \(M\text{?}\)
    Proof..
    Part (a).
    Let \(a,b\in \ann_R(M)\text{.}\) Consider \((a+b)m=am+bm=0\text{.}\) Thus \(a+b\in \ann_R(M)\text{.}\) Let \(r\in\ann_R(M)\) and consider \(-r\text{.}\) Let \(m\in M\) and suppose \(-rm=n\) for some \(n\in R\text{.}\) Add \(rm\) to both sides to see that \(0=n+rm=n\text{.}\) Thus \(n=0\) and \(-rm=0\) for all \(m\in M\text{.}\) So \(-r\in\ann_R(M)\text{.}\)
    Note that as \(0m=0=m0\) for all \(m\in M\text{,}\) we know that \(0\in\ann_R(M)\text{.}\) As \(rm=0\) for all \(m\in M\text{,}\) we see that \(\ann_R(M)\) is a left sided ideal.
    Suppose \(mr=n\) for some \(m\in R\text{.}\) This time we add \(-rm\) to both sides, but as \(-r\in\ann_R(M)\text{,}\) we once again find that \(mr=0\text{.}\) Notice that this means \(rm=mr\) for all \(r\in \ann_R(M)\) and \(m\in M\text{,}\) and thus that elements of \(\ann_R(M)\) commute with elements of \(M\text{.}\)
    Let \(a,b\in \ann_R(M)\text{.}\) Consider \(m(a+b)=ma+mb\text{.}\) Luckily, we know \(ma=am=0=bm=mb,\) and thus that \(ma+mb=0+0=0\text{.}\) Hence \(a+b\in \ann_R(M)\text{,}\) making \(\ann_R(M)\) is a two sided ideal.
    Part (b).
    By the FTFGAG and Sunzi‚Äôs Remainder Theorem, there are only so many options we have for \(M\text{:}\) - \(\Z_{16}\times \Z_{25}\text{,}\) - \(\Z_{8}\times \Z_{2}\times \Z_{25}\text{,}\) - \(\Z_{4}\times \Z_{4}\times \Z_{25}\text{,}\) - \(\Z_{4}\times \Z_{2}\times \Z_{2}\times \Z_{25}\text{,}\) - \(\Z_{2}\times\Z_{2}\times \Z_{2}\times \Z_{2}\times \Z_{25}\text{,}\) - \(\Z_{16}\times \Z_{5}\times \Z_{5}\text{,}\) - \(\Z_{8}\times \Z_{2}\times \Z_{5}\times \Z_{5}\text{,}\) - \(\Z_{4}\times \Z_{5}\times \Z_{20}\) - \(\Z_{4}\times \Z_{2}\times \Z_{2}\times \Z_{5}\times \Z_{5}\text{,}\) and - \(\Z_{2}\times\Z_{2}\times \Z_{5}\times \Z_{20}\text{.}\) However, as ideals are additive subgroups, we know that \(M\) needs to contain a cyclic subgroup of order \(20\text{.}\) Thus we need only consider decompositions with a \(\Z_{20}\) in them, of which there are exactly two: 1. \(\Z_{4}\times \Z_{5}\times \Z_{20}\) 2. \(\Z_{2}\times\Z_{2}\times \Z_{5}\times \Z_{20}\)
    \(\exe\) ‚Äì Standard Free Module Finitely Generated.
    Let \(R\) be a ring. The standard free \(R\)-module of rank \(n\text{,}\) \(R^n\text{,}\) is finitely generated, since it is generated by \(e_1, \dots, e_n\) where \(e_i := \begin{bmatrix} 0 \\ \vdots \\ 0 \\ 1 \\ 0 \\ \vdots \\ 0 \end{bmatrix}\text{,}\) with a \(1\) in the \(i\)-th position. This holds since given any element \(v = \begin{bmatrix} r_1 \\ \vdots \\ r_n\end{bmatrix}\) of \(R^n\) we have \(v = \sum_i r_i e_i\text{.}\)
    In particular, taking \(n = 1\text{,}\) \(R\) is cyclic as a module over itself, since \(R = R \cdot 1\text{.}\) More generally, for any (two sided) ideal \(I\text{,}\) \(R/I\) is a cyclic left \(R\)-module, generated by \(\ov{1}\text{.}\)
    \(\defn\) ‚Äì Linearly Independent (Module).
    Let \(M\) be an \(R\)-module and let \(A\) be a subset of \(M\text{.}\) The set \(A\) is \(\defnn{linearly independent}\) if whenever \(r_1,\ldots,r_n \in R\) and \(a_1,\ldots ,a_n\) are distinct elements of \(A\) satisfying \(r_1a_1 + \dots + r_na_n = 0\text{,}\) then \(r_1 = \dots = r_n = 0\text{.}\) Equivalently, \(A\) is linearly independent if and only if given any equation of the form \(\sum_a r_a a = 0\) where \(r_a \in R\) for all \(a \in A\) and \(r_a = 0\) for all but a finite number of \(a'\)s, we must have \(r_a = 0\) for all \(a\text{.}\)
    \(\defn\) ‚Äì Linearly Dependent (Module).
    Let \(M\) be an \(R\)-module and let \(A\) be a subset of \(M\text{.}\) The set \(A\) is linearly dependent if and only if there is an equation of the form \(\sum_a r_a a = 0\) where \(r_a \in R\) for all \(a \in A\text{,}\) \(r_a = 0\) for all but a finite number of \(a'\)s, and \(r_a \ne 0\) for at least one \(a\text{.}\)
    \(\rem\).
    The empty subset of any module is linearly independent (vacuously).
    \(\exe\) ‚Äì One Element Subsets of \(R\)-Modules.
    A one element subset \(\{m\}\) of an \(R\)-module \(M\) is linearly independent if and only if whenever \(rm =0\text{,}\) we have \(r = 0\text{.}\)
    But it is possible for one elements subsets to be linearly dependent: For example, let \(R\) be any ring and \(I\) and (two-sided) ideal such that \(I \ne 0\text{.}\) Then I claim that every non-empty subset of \(M = R/I\) is linearly dependent. For say \(A\) is a such a nonempty subset. For any \(a \in A\text{,}\) pick any \(r \in I\) such that \(r \ne 0\text{.}\) Then \(ra = 0\) (since \(a = y + I\) for some \(y \in R\) and hence \(ra = ry + I = I = 0_M\)) and this shows \(A\) is linearly dependent. In particular, even a one-element subset of \(M\) is linearly dependent.
    \(\exe\) ‚Äì \(\{3\}\) Linearly Independent in \(\Z\).
    The singleton \(\{3\}\) is a linearly independent subset of the \(\Z\)-module \(M = \Z\text{.}\) But it does not generate all of \(M\text{.}\) The subset \(\{3, 5\}\) does generate all of \(M\text{,}\) but it is not linearly independent, since \(5 \cdot 3 + (-3) \cdot 5 = 0\text{.}\) More on this later.
    \(\defn\) ‚Äì Basis (Module).
    A subset \(A\) of an \(R\)-module \(M\) is a \(\defnn{basis}\) of \(M\text{,}\) if the set \(A\) generates \(M\) and is linearly independent.
    \(\defn\) ‚Äì Free Module.
    An \(R\)-module M is a \(\defnn{free}\) \(R\)-module if \(M\) admits at least one a basis.
    \(\defn\) ‚Äì Rank (Module).
    Let \(R\) be a non-zero commutative ring and let \(M\) be a free \(R\)-module. The cardinality of any basis of \(M\) is called the \(\defnn{rank}\) of \(M\text{.}\)
    \(\exe\) ‚Äì Zero Module is Free.
    The zero module is free with \(\emptyset\) as (its only) basis. This holds since the empty set is vacuously linearly independent and it generates \(\{0\}\text{.}\)
    \(\exe\) ‚Äì Ring is Free Module over Itself.
    \(R\) is free since \(\{1\}\) is a basis for \(R\text{.}\) It clearly generates and if \(r \cdot 1 = 0\) then \(r = 0\text{,}\) so it is linearly independent.
    \(\exe\) ‚Äì \(R^n\) is Free.
    More generally, \(R^n\) is free since \(e_1, \dots, e_n\) is a basis. This is called the \(\defnn{standard basis}\) of \(R^n\text{.}\) We‚Äôve already seen that \(e_1, \dots, e_n\) generates \(R^n\) as an \(R\)-module. Suppose \(\sum r_i e_i = 0\text{.}\) Then \(\begin{bmatrix} r_1 \\ \vdots \\ r_n \end{bmatrix} = \begin{bmatrix} 0 \\ \vdots \\ 0 \end{bmatrix}\) and hence \(r_i = 0\) for all \(i\text{.}\)
    \(\exe\) ‚Äì Quotient of Proper Nonzero Ideals Not Free.
    For any ring \(R\text{,}\) if \(I\) is a (two-sided) ideal such that \(I \ne 0\) and \(I \ne R\text{,}\) then \(M = R/I\) is not free. Since \(I \ne R\text{,}\) \(R/I\) is not the zero module and hence the empty set isn‚Äôt a basis. Let \(A\) be any non-empty subset. Then since \(I \ne 0\text{,}\) as shown above \(A\) is linearly dependent. We conclude that no subset of \(M\) is a basis.
    \(\exe\) ‚Äì Bases are not Unique.
    As you likely know from a class in linear algebra, bases of free modules are rarely unique. Indeed, if \(R\) is any ring, then any single unit forms a basis for \(R\) as a module over itself. For another example, for any ring \(R\) and any fixed element \(r \in R\text{,}\) the set
    \begin{equation*} \left\{ \begin{bmatrix} 1 \\ 0 \end{bmatrix}, \begin{bmatrix} r \\ 1 \end{bmatrix} \right \} \end{equation*}
    forms a basis for the free \(R\)-module \(R^2\text{.}\)

    Let \(R\) be a commutative ring with \(1 \neq 0\text{.}\) 1. Show that if \(M\) is a free \(R\)-module, then \(\ann(M) = 0\text{.}\) 2. Give an example of a ring \(R\) an a nonzero module \(M\) such that \(\ann(M) \neq 0\text{.}\)
    \(\rem\).
    A key difference between free modules over rings that are not fields and vector spaces is that not every linearly independent subset of a free module can be extended to a basis. For example, \(\{3\}\) is a linearly independent subset of \(\Z\text{,}\) but it cannot be extended to a basis. Indeed, any set of the form \(\{3, n \}\) with \(n \ne 3\) is linearly dependent since \(n \cdot 3 + (-3) \cdot n = 0\text{.}\)
    Likewise, over arbitrary rings, not every subset that generates a free module necessarily contains a basis. For instance, \(\{3,5\}\) generated \(\Z\) as a module over itself, but no subset of it is a basis.
    (Note that \(\Z\) has precisely two bases as a module over itself: \(\{1\}\) and \(\{-1\}\text{.}\) )
    \(\rem\).
    If \(R\) is the zero ring, then there is (up to isomorphism) only one \(R\)-module, the zero module. (Proof: If \(M\) is any module over the \(0\) ring, then for each \(m \in M\) we have \(m = 1_R \cdot m = 0_R \cdot m = 0_M\text{.}\)) Checking the definition carefully, we see that both the empty set and the set \(\{0\}\) form bases for the zero module over the \(0\) ring. This gives an example of a module with two bases of different cardinalities. On the homework you will encounter another example of this sort.
    \(\lem\) ‚Äì Elements Uniquely Expressible in Free Modules.
    Suppose \(M\) is a free \(R\)-module and \(A\) is a basis of \(M\text{.}\) Then every element of \(M\) is uniquely expressible as an \(R\)-linear combination of elements of \(A\text{.}\)
    More precisely, for each \(m\) there is unique family of elements \((r_a)_{a \in A}\text{,}\) with \(r_a \in R\) for all \(a\text{,}\) such that \(r_a = 0\) for all but a finite number of indices \(a \in A\) and \(m = \sum_{a \in A} r_a a\text{.}\)
    Proof..
    Given \(m \in M\text{,}\) the fact that
    \begin{equation*} m = \sum_a r_a a \end{equation*}
    for at least one family of elements \((r_a)_{a \in A}\) with \(r_a = 0\) for all but a finite number of \(a\)‚Äôs is the definition of what it means for \(A\) to generate \(M\text{.}\)
    Suppose \((s_a)_{a \in A}\) is another such family with \(m = \sum_a s_a a\text{.}\) Then
    \begin{equation*} 0 = m-m = \sum_a r_a a - \sum_a s_a a = \sum_a (r_a - s_a) a. \end{equation*}
    Since \(A\) is linearly independent, by definition \(r_a - s_a = 0\) for all \(a\text{.}\)
    \(\exe\) ‚Äì Module with Infinite Basis.
    I have not yet given an example of a module with an infinite basis; here is one. Let \(R\) be any ring and \(M = R[x]\)[^1] (which, recall, is an \(R\)-module due to the evident ring map \(R \to R[x]\)). Then the countably infinite set \(\{1,x,x^2,\dots, x^n,\dots\}\) is a basis. The fact that this set is a basis is essentially part of the definition of \(R[x]\text{.}\) The Lemma says that every polynomial is uniquely expressible as an \(R\)-linear combination of (a finite subset of) \(1,x,x^2,\dots\text{.}\)
    \(\thm\) ‚Äì UMP for Free \(R\)-Modules.
    Let \(R\) be a ring, let \(M\) be a free \(R\)-module with basis \(B\text{,}\) let \(N\) be an \(R\)-module, and let \(j: B \to N\) be any function. Then there is a unique \(R\)-module homomorphism \(h: M \to N\) such that \(h(b) = j(b)\) for all \(b \in B\text{.}\)
    In other words, there is a bijection of sets
    \begin{equation*} \Hom_R(M,N) \leftrightarrow\Fun(B, N) \end{equation*}
    given by sending a homomorphisms \(f: M \to N\) to its restriction \(f|_B:B \to N\) to \(B\text{.}\) (Here, \(\Hom_R(M,N)\) is the set of all \(R\)-module homomorphisms from \(M\) to \(N\) and \(\Fun(B, N)\) is the set of all functions from \(B\) to \(N\text{.}\))
    Proof..
    {} Given a function \(j: B \to N\text{,}\) define \(f: M\to N\) as follows: Given \(m \in M\text{,}\) by Lemma \(m\) can be written uniquely as a finite sum \(m = \sum_{b \in B} r_b b\text{.}\) We set
    \begin{equation*} f(m) = \sum_{b \in B} r_b j(b). \end{equation*}
    Note that \(f\) is a well-defined function by the uniqueness of the equation \(m = \sum_{b \in B} r_b b\text{.}\)
    We need to prove \(f\) is an \(R\)-module homomorphism. I‚Äôll just show it preserves scaling ‚Äî the proof for addition is similar. Given \(x \in R\) and \(m \in M\text{,}\) we have \(m = \sum_{b \in B} r_b b\) for some \(r_b \in R\text{,}\) and hence \(xm = \sum_b (xr_b) b\text{.}\) By definition of \(f\text{,}\)
    \begin{equation*} f(xm) = \sum_b (xr_b) j(b) = x \sum_b r_b j(b) = x f(m). \end{equation*}
    Finally, for any \(y \in B\) we have \(y = \sum_{b \in B} r_b b\) where \(r_b = 1\) if \(b = y\) and \(r_b = 0\) if \(b \ne y\text{.}\) So \(f(y) = j(y)\) by construction. This proves existence.
    {} Let \(g:M\to N\) be another \(R\)-module homomorphism such that \(g(b)=g(b)\) for each \(b \in B\text{.}\) Given \(m \in M\) we have \(m = \sum_{b \in B} r_b b\) and hence
    \begin{equation*} g(m) = \sum_b g(r_b (b) = \sum_b r_b g(b) = \sum_b r_b j(b) = f(m). \end{equation*}
    and hence \(g = f\text{.}\)
    \(\rem\).
    The uniqueness only uses that \(B\) generates \(M\) as an \(R\)-module.
    \(\cor\) ‚Äì Free Modules with Equal Basis Elements Isomorphic.
    If \(M\) and \(N\) are free \(R\)-modules having bases of the same cardinality, then \(M\) and \(N\) are isomorphic \(R\)-modules.
    More precisely, if \(A\) is a basis of \(M\) and \(B\) is a basis of \(N\) and \(j:A\to B\) is a bijection of sets, then there is a unique \(R\)-module isomorphism \(h: M \xra{\cong} N\) such that \(h|_A = j\)[^1].
    Proof..
    Let \(h:M\to N\) and \(h':N\to M\) be the \(R\)-module homomorphisms induced by the bijection \(j:A\to B\) and its inverse \(j^{-1}:B\to A\text{,}\) respectively, using the UMP for free modules. We‚Äôll show that \(h\) and \(h'\) are mutual inverses. For this note that \(h'\circ h:N\to N\) is an \(R\)-module homomorphism and \((h'\circ h)(b)=h'(j(b))=j^{-1}(j(b))=b\) for every \(b\in B\text{.}\) Since the identity map \(\id_N\) is also an \(R\)-module homomorphism such that \(id_N(b)=b\) for every \(b\in B\text{,}\) by the uniqueness clause in the UMP, we have \(h'\circ h=\id_n\text{.}\) Similarly \(h\circ h'=\id_M\text{.}\)
    \(\exe\) ‚Äì \(M\cong R^n\) as \(R\)-modules.
    If \(M\) is a free \(R\)-module that has a basis of cardinality \(n\text{,}\) then \(M \cong R^n\) as \(R\)-modules. This holds since, as seen above, \(R^n\) has a basis of cardinality \(n\text{,}\) namely the standard basis \(e_1, \dots, e_n\text{.}\)
    More precisely, as the proof makes clear, if \(B = \{x_1, \dots, x_n\}\) is an (ordered) basis of \(M\text{,}\) there is an isomorphism \(\phi_B: R^n \cong M\) that sends \(\begin{bmatrix} r_1 \\ \vdots \\ r_n \end{bmatrix}\) to \(\sum_i r_i x_i\text{.}\)
    \(\rem\).
    Beware that the cardinality of a basis of a free modules is not an isomorphism invariant in general! There exist rings \(S\) such that \(S^n\) and \(S^m\) are isomorphic \(S\)-modules for all positive integers \(n\) and \(m\text{.}\)
    \(\thm\) ‚Äì Uniqueness of Rank over Commutative Rings.
    Let \(R\) be a commutative ring such that \(1 \ne 0\) and let \(M\) be a free \(R\)-module with bases \(A\) and \(B\text{.}\) Then \(A\) and \(B\) have the same rank, i.e.¬†there exists a (non unique) bijection of sets joining them.
    Proof..
    Let \(R\) be a non-zero commutative ring and let \(M\) be a free \(R\)-module with two {} bases, \(A\) and \(B\text{.}\) We need to show \(A\) and \(B\) have the same cardinality. Since \(A\) and \(B\) are finite, by Example the assertion is equivalent to the following statement:
          \begin{quote} For a non-zero commutative ring $R$,
      if there is an isomorphism $R^n \cong R^m$ of $R$-modules for some integers $n$ and $m$, then $n = m$.
      \end{quote}
    I will prove this statement by taking it as already known that it holds in the special case when \(R\) is a field. (We will prove it for fields later.)
    Since \(R\) is not the zero ring, it contains at least one maximal ideal \(I\text{.}\) (This is proven using Zorn‚Äôs Lemma.) Recall that \(R/I\) is a field.
    Given an isomorphism \(f: R^n \xra{\cong} R^m\) of \(R\)-modules, by Lemma we have an induced homomorphism of \(R/I\)-module \(\ov{f}: R^n/IR^n \to R^m /IR^m\text{.}\) Likewise, the inverse map \(f^{-1}: R^m \xra{\cong} R^n\) induces a map \(\ov{f^{-1}}: R^m/IR^m \to R^n /IR^n\text{.}\) Also by that Lemma we have \(\ov{f} \circ \ov{f^{-1}} = \ov{f \circ f^{-1}} = \ov{\id} = \id\) and similarly \(\ov{f^{-1}} \circ \ov{f}\) is the identity. That is, we have an isomorphism \(R^n/IR^n \cong R^m /IR^m\) of \(R/I\)-modules.
    Next, I claim that there is an isomorphism
    \begin{equation*} R^n/IR^n \cong (R/I)^n \end{equation*}
    of \(R/I\)-modules. Define \(R^n \to (R/I)^n\) in the evident way (modding out by \(I\) entry-wise). It is a surjective map of \(R\)-modules with kernel \(I R^n\) and thus, by the First Isomorphism Theorem, it induces an isomorphism
    \begin{equation*} g: R^n/I R^n \xra{\cong} (R/I)^n \end{equation*}
    given by \(g((r_1, \dots, r_n)^\tau + IR^n) = (r_1 + I, \dots, r_n + I)^\tau\) (where \(\tau\) denotes taking the transpose).Now, what we have said so far only shows that \(g\) is isomorphism of \(R\)-modules, but it is easy to see that \(g\) is in fact \(R/I\)-linear (I‚Äôll leave that to you) and thus it is an isomorphism of \(R/I\)-modules.
    Putting the results proven so far together, we conclude that \((R/I)^n\) and \((R/I)^m\) are isomorphic as \(R/I\)-modules. Since \(R/I\) is a field and since we are assuming the result holds for fields, we deduce that \(n = m\text{.}\)
    \(\prop\) ‚Äì Bijection of Matrices and Hom Functor.
    Suppose \(R\) is a ring and \(F\) and \(E\) are free \(R\)-modules with ordered bases \(X = \{x_1, \dots, x_m\}\) and \(Y = \{y_1, \dots, y_n\}\text{,}\) respectively. There is a bijective correspondence
    \begin{equation*} \Mat_{n \times m}(R) \to \Hom_R(F, E) \end{equation*}
    that sends a matrix \(A = (a_{i,j})_{1 \leq i \leq n, 1 \leq j \leq m}\) to the unique \(R\)-module homomorphism \(T_A: F \to E\) satisfying \(T_A(x_j)=\sum_{i= 1}^n a_{i,j} y_i\) for each \(1 \leq j \leq m\text{.}\)

    It is of paramount importance to realize that the bijection between maps and matrices {}.
    In particular, the \(R\)-map written as \(T_A\) that we associate to the matrix \(S\) in this proposition depends on the basis \(X\) and \(Y\text{,}\) and so it should really be written as something like \(T_A^{X,Y}\text{.}\) I will occasionally do so.

    commutes, where \(\phi^F_X\) and \(\phi^E_Y\) are defined in and and by \(A: R^m \to R^n\) we mean the map \(v \mapsto Av\) where \(Av\) denotes the usual rule for matrix multiplication. (Recall elements of \(R^m\) are represented as column vectors.)
    Conversely, given a matrix \(A\text{,}\) the corresponding map \(T_A^{X,Y}: F \to E\) is the unique one causing this square to commute; i.e.¬†\(T_A^{X,Y} = \phi_Y^E \circ A \circ (\phi^F_X)^{-1}\text{.}\)

    \(\lem\) ‚Äì Matrix Multiplication is Associative.
    Let \(R\) be a commutative ring, let \(F, E, G\) be free \(R\)-modules having finite bases \(X = \{x_1, \dots, x_m\}, Y = \{y_1, \dots, y_n\}, Z = \{z_1, \dots, z_p\}\) of size \(m,n,p\text{,}\) respectively. Given a \(n \times m\) matrix \(A\) and a \(p \times n\) matrix \(B\text{,}\) let \(AB\) denote the \(p \times m\) matrix obtained by the usual formula for matrix multiplication, and let \(T_A^{X,Y}: F \to E\text{,}\) \(T_B^{Y,Z}: E \to G\) and \(T_{BA}^{X,Z}: F \to G\) be the maps associated to these matrices relative to the specified bases, as given in Proposition . Then
    \begin{equation*} T^{Y,Z}_B \circ T_A^{X,Y} = T_{BA}^{X,Z}. \end{equation*}
    In particular, matrix multiplication is associative. ###### Proof. For the first assertion we just need to check the two maps agree on \(x_j\) for all \(j\text{:}\) We have
    \begin{equation*} T_B(T_A(x_j)) = T_B(\sum_i a_{i,j} y_i) = \sum_i a_{i,j} T_B(y_i) = \sum_i \sum _l a_{i, j} b_{k, i} z_k \end{equation*}
    (with the second equation using that \(T_B\) is an \(R\)-map) and
    \begin{equation*} T_{BA}(x_j) = \sum_k (BA)_{k,j} z_k = \sum_k \left(\sum _i b_{k, i} a_{i, j}\right) z_k. \end{equation*}
    (I have left off the superscripts \((X,Y)\) etc. for the sake of my sanity.) These two expressions agree since \(R\) is commutative.
    The second assertion holds since composition of functions is associative.
    \(\rem\).
    The correct version of this for \(R\)-non commutative is that \(T_B \circ T_A = T_{(A^\tau B^\tau)^\tau}\) (again, with superscripts omitted).

    Problem 5 ‚Äì Maximal Linearly Independent.
    Let \(R\) be a commutative integral domain and \(M\) an \(R\)-module. Recall that a subset \(S\) of \(M\) is called a maximal linearly independent set of \(M\) if \(S\) is linearly independent and any subset of \(M\) properly containing \(S\) is linearly dependent.
    1. Let \(T\) be a linearly independent subset of \(M\text{.}\) Prove that \(T\) is contained in some maximal linearly independent subset of \(M\text{.}\)
    2. Let \(T\) be a linearly independent subset of \(M\) and \(N\) the \(R\)-submodule of \(M\) generated by \(T\text{.}\) Prove that \(T\) is a maximal linearly independent subset if and only if \(M/N\) is torsion. (Recall that an \(R\)-module \(P\) is called ‚Äútorsion‚Äô‚Äô if for each \(p \in P\text{,}\) there is a \(r \in R\) such that \(r \ne 0\) and \(rp = 0\text{.}\))
    Proof..
    Part (a).
    Let \(A\) be the set of all linearly independent subsets of \(M\) that contain \(T\text{.}\) We can order \(A\) with respect to inclusion. Let \(X\) be a totally ordered subset of \(A\text{,}\) and let \(U\) be the union of all elements in \(X\text{.}\) Let \(\{u_i\}\) be a set of elements in \(U\) such that \(\sum_{i=0}^nr_iu_i=0\) for some \(r_i\in R\text{,}\) where \(i\leq n\) for some \(n\in\N\text{.}\) As \(U\) is the union of all elements in \(X\text{,}\) there exists some \(B_1\) such that \(u_1\in B_1\text{.}\) However, as \(X\) is totally ordered, there exists some \(B_2\) such that \(B_2\) contains \(B_1\) and \(u_2\in B_2\text{.}\) Continuing in this way, we see that there exists some \(B_n\in X\) such that \(\{u_i\}\subseteq B_n\text{.}\) As \(B_n\) is linearly independent, we know that \(\sum_{i=0}^nr_iu_i=0\) means that \(r_i=0\) for all \(i\text{.}\) Thus \(U\) is indeed linearly independent, making it an upper bound for \(X\text{.}\) Thus by Zorn‚Äôs Lemma there exists a maximal element of \(A\text{,}\) which we denote \(S\text{.}\) Thus \(S\) is linearly independent, contains \(T\text{,}\) and is maximal.
    Part (b).
    \((\Rightarrow)\) Suppose \(T\) is maximal linearly independent, and suppose by way of contradiction that \(M/N\) is not torsion. Thus there exists some \(xN\in M/N\) such that for all \(r\in R\text{,}\) we see that \(rx\neq0\text{.}\) However, as \(x\neq 0\) and \(x\not\in N\text{,}\) this means that \(x\not\in T\text{.}\) Consider \(T\cup\{x\}\text{.}\) This set is linearly independent, contradicting the assumption that \(T\) was maximal.
    \((\Leftarrow)\) Suppose \(M/N\) is torsion. Let \(x\neq0\in M\) and consider \(T\cup\{x\}\text{.}\) Consider \(x+N\in M/N\text{.}\) As \(M/N\) is torsion, there exists an \(r\neq0\in R\) such that \(rxN=0\text{.}\) Thus \(rx\in N\text{.}\) (Note, if \(x\in N\text{,}\) then \(r=1\)). As \(rx\in N\) and \(N\) is generated by \(T\) (\(N=RT\)), \(rx=\sum r_it_i\text{.}\) Subtracting over we see that \(rx-\sum r_it_i=0\text{.}\) But as \(r\neq 0\text{,}\) we see that each \(t_i\) and \(x\) are in \(T\cup\{x\}\text{,}\) but the sum is 0. Thus \(T\cup\{x\}\) is linearly dependent.
