    Section 9.1 Ring Basics

    Subsection From Rings to Fields

    ‚ÄúI made a lot of mistakes out of the ring, but I never made any in it.‚Äù
    ‚ÄïJack Johnson

    Definition 9.1. Ring.

    A ring is a set \(R\) equipped with two binary operations, \(+\) and \(\cdot\text{,}\) satisfying:
    • \((R,+)\) is an abelian group with identity element denoted \(0\text{,}\)
    • Associative Multiplication.
      \(\cdot\) is associative (making \((R,\cdot)\) a )
    • Distributive Law.
      \(a \cdot (b + c) = a \cdot b + a \cdot c\) and \((a + b) \cdot c = a \cdot c + b \cdot c\) hold for all \(a,b,c \in R\text{.}\)
    Here are some basic consequences of the axioms.

    Proposition 9.2. Ring Arithmetic.

    For any ring \(R\) and all \(a,b \in R\) we have:
    1. \(a \cdot 0 = 0 = 0 \cdot a\text{,}\)
    2. \((-a)b = -(ab) = a(-b)\text{,}\)
    3. \((-a)(-b) = ab\text{.}\)

    Proof.

    Let \(R\) be a ring with identity.
    1. Let \(a\in R\text{.}\) Observe that \(0a=(0+0)a=0a+0a\) by the distributive law. Subtracting \(0a\) from both sides we have \(0=0a\text{.}\) A similar argument shows that \(0=a0\text{.}\)
    2. Let \(a\in R\text{.}\) Note that \(-a+a=0\text{.}\) Consider \((-1)a+a\text{.}\) Using the distributive law we see that \((-1)a+a=a(-1+1)=a(0)=0\text{.}\) Thus \(-a\) and \((-1)a\) are both additive inverses of \(a,\) making them unique.

    Definition 9.3. Unital Ring.

    A ring \(R\) is a unital ring (or a ring with identity) if there is a multiplicative identity element, written as \(1\text{,}\) such that \(1 \cdot a = a = a \cdot 1\) for all \(a \in R\text{.}\)

    Remark 9.4.

    Around these parts most of us generally include \(1\) in our definition of a ring, referring to a ring without an identity as a rng ‚Äâ1‚Äâ. However, we'll stick with the more classical definition for now.
    Some additional consequences.

    Proposition 9.5. Unital Ring Arithmetic.

    For a unital ring \(R\) and all \(a\in R\) we have:
    1. \(1\) is unique, and
    2. \((-1)a = -a\text{.}\)
    We can go deeper.

    Definition 9.6. Division Ring.

    A unital ring \(R\) is a division ring if \(1\neq 0\) and \((R \sm \{0\})\) is a group under \(\cdot\) (i.e. every \(r\in (R \sm \{0\})\) has a multiplicative inverse)
    We can go deeper.

    Definition 9.7. Commutative Ring.

    A ring \(R\) is commutative if \(a \cdot b = b \cdot a\) holds for all \(a,b \in R\text{.}\)

    Discussion 9.1.

    One could argue that commutative rings should really be called abelian rings to remain consistent with abelian groups. Discuss.
    We can go deeper.

    Definition 9.8. Field.

    A ring \(R\) is a field if it is a commutative division ring.
    Phew. That was pretty deep. Let's take a breather and see some examples.

    Example 9.9. Relevant Rings.

    1. Trivial Ring.
      \(R=\{0\}\) is called the trivial ring, or zero ring.‚Äâ2‚Äâ
    2. \(\mathbb{Z}\) is a commutative ring.
    3. \(\mathbb{Z}/n\) is a commutative ring under addition and multiplication modulo \(n\text{.}\)
    4. The familiar sets \(\mathbb{Q}, {\mathbb{R}}, \mathbb{C}\) are fields.
    5. Gaussian Integers.
      The set \(\Z[i] = \{a + bi \mid a,b \in \Z\}\) is a commutative ring known as the Gaussian integers.
    6. Matrix Ring.
      If \(R\) is any ring (not necessarily commutative), so is \(\operatorname{M}_{n}(R)\) for any natural number \(n\text{,}\) using the usual rules for addition and multiplication of square matrices.
    7. Direct Product of Rings.
      The cartesian product \(R\times R'\) of two rings \(R\) and \(R'\) has a natural ring structure with addition and multiplication defined componentwise:
      \begin{equation*} (a,b)+(c,d)=(a+c,b+d) \end{equation*}
      \begin{equation*} (a,b)\cdot(c,d)=(a\cdot c,b\cdot d) \end{equation*}

    Exploration 9.10. \(1=0\).

    Let \(R\) be a ring. Prove that \(R=\{0\}\) if and only if \(1=0\text{.}\)
    Solution.
    Notice that in the trivial ring \(0=1\text{.}\) Conversely, if \(1 = 0\) in a ring, then \(R = \{0\}\text{,}\) since in this case for all \(a\text{,}\) we have \(a \cdot 0 = 0\) and hence \(a = a \cdot 1 = a \cdot 0 = 0\text{.}\)

    Exploration 9.11. Direct Product Preservations.

    Let \(R\) and \(S\) be rings. The Cartesian product \(R\times S\) is commutative if and only if \(R\) and \(S\) are commutative. Similarly, \(R\times S\) has identity if and only if both \(R\) and \(S\) do as well.

    Example 9.12. Less Relevant Rings.

    1. Opposite Day.
      Given a ring \(R\text{,}\) let \(R^{op}\) denote the ‚Äúopposite ring''. This is the same underlying set as \(R\) equipped with the same rule for \(+\) as \(R\text{,}\) but with multiplication rule (which I will write here as \(\cdot_{op}\)) redefined to be \(a \cdot_{op} b := b a\) (where \(ba\) refers to the original multiplication rule for \(R\)). Then \(R^{op}\) is also a ring
    2. The Real Hamiltonian Quaternion Ring.
      Let \(i,j,k\) be formal symbols and set \({\mathbb{H}}\) to be the four dimensional \({\mathbb{R}}\)-vector space consisting of all expressions of the form \(a + bi + cj + dk\) with \(a,b,c,d \in {\mathbb{R}}\text{.}\) Addition is vector space addition:
      \begin{equation*} (a + bi + cj + dk) +(a' + b'i + c'j + d'k) =(a') + (b + b') i + (c + c')j + (d + d')k. \end{equation*}
      Multiplication is uniquely determined by the axioms of a ring together with the rules
      \begin{equation*} i^2 = j^2 = k^2 = -1, -ji = ij = k, -kj = jk = i, -ik = ki = j. \end{equation*}
      and the fact that the real coefficients commute with each other and \(i,j,k\text{.}\)
      It's not obvious that the multiplication defined in this way satisfies associativity, but in fact it does (this amounts conditions very similar to the associativity of the group \(Q_8\)).
      \({\mathbb{H}}\) is a division ring, since one can check that
      \begin{equation*} (a + bi + cj + dk)^{-1} = \frac{a - bi - cj - dk}{\|a + bi + cj + dk\|} \end{equation*}
      where
      \begin{equation*} \|a + bi + cj + dk \| := a^2 + b^2 + c^2 + d^2. \end{equation*}
      In the equation above \(\|a + bi + cj + dk \|\) is non-zero real number if \(a + bi + cj + dk\) is not the zero element. The quantity \(\|a + bi + cj + dk \|\) is called the norm of the quaternion \(a + bi + cj + dk\text{.}\)
    3. Function Ring.
      If \(X\) is a set and \(R\) is a ring, let \(\operatorname{Fun}(X, R)\) be the collection of set theoretic functions from \(X\) to \(R\text{,}\) and define \((f + g)(x) = f(x) + g(x)\) and \((f \cdot g)(x) := f(x) \cdot g(x)\text{.}\) Then \(\operatorname{Fun}(X,R)\) is a ring. If \(X\) is a finite set and \(|X|=n\text{,}\) then \(\operatorname{Fun}(X,R)\) may be identified with \(R^n=\underbrace{R \times \cdots \times R}_n\text{,}\) the direct product of \(n\) copies of \(R\text{.}\)
    4. Endomorphism Ring.
      If \(A = (A, +)\) is any abelian group, set \(\operatorname{End}_{Ab}(A)\) to be the collection of endomorphisms of \(A\) ‚Äî that is, the set of group homomorphisms \(f: A \to A\) from \(A\) to itself.
      Then \(\operatorname{End}_{Ab}(A)\) is a ring with addition \((f + g)(a) := f(a) + g(a)\) and multiplication \(f \cdot g := f \circ g\text{.}\) This is almost always a non-commutative ring.
    Here is a nice generalization that will prove usful quite a few times down the road.

    Theorem 9.13. Binomial Theorem for Commutative Rings.

    For any commutative ring \(R\) and any elements \(a\) and \(b\) in \(R\text{,}\) we have:
    \begin{equation*} (a+b)^n = \sum_{i=0}^n {n\choose i} a^{n-i}b^i \end{equation*}
    where \({n\choose i}\) is the binomial coefficient, defined as: \({n\choose i} = \frac{n!}{i!(n-i)!}\) with integers \(n\) and \(i\) such that \(0\le i\le n\text{.}\)
    Get it?
    Personally, I am in favor of calling this the bor-ring.
