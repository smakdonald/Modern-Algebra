    Chapter 25 Simple Modules

    Section 25.1 Simple Modules

    Now we proceed to discuss some smallness conditions on modules. The first key notion is that of a simple module. Simple modules are the atoms in module theory.

    Definition 25.1.

    An \(R\)-module \(M\) is simple if there are no nonzero proper submodules of \(M\text{.}\)

    Lemma 25.2.

    Let \(M\) be a nonzero \(R\)-module. The following are equivalent:
    1. \(M\) is simple
    2. \(Rm= M\) for all \(m\in M\smallsetminus 0\)
    3. \(M\cong R/I\) for some maximal left ideal \(I\text{.}\)

    Proof.

    For a left ideal \(I\text{,}\) the submodules of \(R/I\) are in bijective correspondence with the left \(R\)-submodules of \(R\) that contain \(I\text{,}\) i.e., the left ideals that contain \(I\text{.}\) It is then clear that if \(I\) is a maximal left ideal, then \(R/I\) is simple, so (3) implies (1). On the other hand, if \(M\) is simple then it is cyclic (since (1) implies (2)), so \(M\cong R/I\) for some left ideal \(I\text{,}\) and if \(I \subsetneqq J\) for some proper left ideal \(J\text{,}\) then \(0 \neq J/I \subsetneqq R/I\text{;}\) thus (1) implies (3).

    Example 25.3.

    1. If \(K\) is a field, a \(K\)-vector space is simple if and only if it is \(1\)-dimensional. Moreover, if \(R\) is a \(K\)-algebra, then any \(R\)-module that is \(1\)-dimensional as a vector space is a simple \(R\)-module as well.
    2. If \(R\) is commutative, then an \(R\)-module \(M\) is simple if and only if \(M\) is isomorphic to a field.
    3. Let \(R={\mathbb{R}}[D_{2n}]\text{,}\) and \(V\) be the natural \(2\)-dimensional representation by reflections and rotations. Then \(V\) is a simple \(R\)-module, since there are no \(D_{2n}\)-stable subspaces.
    4. Let \(K\) be a field, or more generally a division ring, and let \(R=M_n(K) \cong \operatorname{End}_K(K^n)\text{.}\) The module \(M=K^n\) of column vectors is a simple \(R\)-module Indeed, if \(v=(a_1,\dots,a_n)\neq 0\text{,}\) say \(a_i\neq 0\text{;}\) then \(a_i^{-1} E_{ij} v = e_i \in M\text{,}\) and since \(M\) is generated by the standard vectors \(e_i\text{,}\) \(M = Rv\text{.}\)

    Lemma 25.4. Schur's Lemma.

    Let \(R\) be a ring, and \(M,N\) be two simple \(R\)-modules. Then every nonzero \(R\)-module homomorphism \(\phi:M\to N\) is an isomorphism. In particular, \(\operatorname{End}_R(M)\) is a division ring.

    Proof.

    For the first assertion, let \(f:M\to N\) be \(R\)-linear and nonzero. Then \(\operatorname{ker}(f)\neq M\text{,}\) so \(\operatorname{ker}(f)=0\) by simplicity, and \(\operatorname{im}(f) \neq 0\text{,}\) so \(\operatorname{im}(f) = N\text{.}\)
    For the second, recall that \(\operatorname{End}_R(M)\) is a ring. If \(f\in \operatorname{End}_R(M)\) is nonzero, then by the first part, it is an isomorphism, so it has a two-sided inverse in \(\operatorname{End}_R(M)\text{.}\)

    Section 25.2 Finite Length Modules

    Definition 25.5.

    Given a short exact sequence
    \begin{equation*} 0 \to A \to B \to C \to 0 \end{equation*}
    we may think of the middle module \(B\) as built out of \(A\) and \(C\text{;}\) we call \(B\) an extension of \(A\) and \(C\text{.}\) Suppose that a module has a finite sequence of submodules
    \begin{equation*} 0 = M_0 \subseteq M_1 \subseteq M_2 \subseteq \cdots \subseteq M_n = M \end{equation*}
    we call such a sequence a filtration. Then \(M_1\) is an extension of \(M_0\) and \(M_1/M_0\text{,}\) \(M_2\) is an extension of \(M_1=M_1/M_0\) and \(M_2/M_1\text{,}\) and so on. We might think of \(M\) as built from \(M_1/M_0, M_2/M_1,\dots,M_n/M_{n-1}\) like so.

    Definition 25.6.

    A module \(M\) has finite length if it has a filtration of the form
    \begin{equation*} 0 = M_0 \subseteq M_1 \subseteq M_2 \subseteq \cdots \subseteq M_n = M \end{equation*}
    with \(M_{i+1}/M_i\) simple for each \(i\text{;}\) such a filtration is called a composition series of length \(n\text{.}\) We say a composition series is strict if \(M_i \neq M_{i+1}\) for all \(i\text{.}\) Two composition series are equivalent if the collections of composition factors \(M_{i+1}/M_i\) are the same up to reordering. The length of a finite length module \(M\text{,}\) denoted \(\ell(M)\) , is the minimum of the lengths of a composition series of \(M\text{.}\) If \(M\) has does not have finite length, we say that \(M\) has infinite length, or \(\ell(M) = \infty\text{.}\)

    Example 25.7.

    Let \(K\) be a field and \(V=K^2\text{.}\) Then any filtration of the form \(0 \subseteq W \subseteq V\) where \(W\) is a line through the origin is a strict composition series.

    Remark 25.8.

    Let
    \begin{equation*} 0 \to A \xrightarrow{i} B \xrightarrow{p} C\to 0 \end{equation*}
    be a short exact sequence. Given filtrations / composition series / strict composition series
    \begin{equation*} A_\bullet: \qquad 0 = A_0 \subseteq A_1 \subseteq A_2 \subseteq \cdots \subseteq A_n = A \end{equation*}
    and
    \begin{equation*} C_\bullet: \qquad 0 = C_0 \subseteq C_1 \subseteq C_2 \subseteq \cdots \subseteq C_n = C \end{equation*}
    we can make a filtration / composition series / strict composition series of \(B\) by
    \begin{equation*} 0 = i(A_0) \subseteq i(A_1) \subseteq i(A_2) \subseteq \cdots \subseteq i(A_n) = i(A) = p^{-1}(C_0) \subseteq p^{-1}(C_1) \subseteq p^{-1}(C_2) \subseteq \cdots \subseteq p^{-1}(C_n) = B. \end{equation*}
    Conversely, given a filtration / composition series / strict composition series of \(B\) that contains \(i(L)\) as a term, we can obtain filtrations / composition series / strict composition series of \(A\) and \(C\) by applying \(i^{-1}\) to the terms up through \(i(L)\) and applying \(p\) to the terms from \(i(L)\) on. However, not every filtration / composition series of a module will contain a fixed submodule as a term.

    Theorem 25.9. JOrdan-Holder Theorem.

    Let \(M\) be a module of finite length.
    1. If \(L\subseteq M\) is a proper submodule, then \(\ell(L) < \ell(M)\text{.}\)
    2. If \(L\subseteq M\) is a nonzero submodule and \(\overline{M}=M/L\text{,}\) then \(\ell(\overline{M})< \ell(M)\text{.}\)
    3. Any filtration of \(M\) can be refined to a composition series.
    4. All strict composition series for \(M\) are equivalent, and hence have the same length.

    Proof.

    If \(m := \ell(M)\text{,}\) consider a strict composition series of \(M\) of length \(m\text{,}\) say
    \begin{equation*} M_\bullet: \qquad 0 = M_0 \subsetneqq M_1 \subsetneqq M_2 \subsetneqq \cdots \subsetneqq M_m = M. \end{equation*}
    1. Consider the filtration
      \begin{equation*} L_\bullet: \qquad 0 = M_0 \cap L \subseteq M_1 \cap L \subseteq M_2 \cap L \subseteq \cdots \subseteq M_m \cap L = L. \end{equation*}
      By the Second Isomorphism Theorem, its composition factors satisfy
      \begin{equation*} \frac{M_{i+1} \cap L}{M_i \cap L} = \frac{M_{i+1} \cap L}{(M_{i+1} \cap L) \cap M_i} \cong \frac{M_{i+1} \cap L + M_i} {M_i}. \end{equation*}
      The right hand side is a submodule of \(M_{i+1}/M_i\text{,}\) which by assumption is simple, so our filtration is in fact a composition series of length \(n\text{.}\) Then for any \(i\) either
      \begin{equation*} \frac{M_{i+1} \cap L}{M_i \cap L} = 0 \quad \textrm{ or } \quad \frac{M_{i+1} \cap L}{M_i \cap L}\cong \frac{M_{i+1}}{M_i}. \end{equation*}
      We claim that the latter case does not hold for all \(i\text{:}\) if it did, we would have \(0=M_0 = M_0 \cap L\text{,}\) and inductively \(M_{i+1} \cap L = M_{i+1}\) for all \(i\) and in particular for \(i=m-1\text{,}\) we have \(M = M\cap L\text{,}\) contradicting that \(L\) is proper. Thus, for some \(i\text{,}\) the first case holds. We can then skip that \(i\) and obtain a composition series of length less than \(n\text{,}\) so \(\ell(L)<m\text{.}\)
    2. Consider the filtration
      \begin{equation*} \overline{M}_\bullet: \qquad 0 = \frac{M_0 + L }{L} \subseteq \frac{M_1 + L }{L} \subseteq \cdots \subseteq \frac{M_n + L }{L} = \overline{M}. \end{equation*}
      The factors satisfy
      \begin{equation*} \frac{(M_{i+1} + L )/ L }{(M_i + L) / L} \cong \frac{M_{i+1} + L}{M_{i} + L} \cong \frac{M_{i+1} + (M_i+L)}{M_i+L}\cong \frac{M_{i+1} }{M_{i+1} \cap( M_i + L)}, \end{equation*}
      and since \(M_i \subseteq M_{i+1} \cap (M_i+L)\text{,}\) these are quotient modules of the simple module \(M_{i+1}/M_i\text{,}\) so this is a composition series. Then for any \(i\) either
      \begin{equation*} \frac{(M_{i+1} + L )/ L }{(M_i + L) / L} = 0 \quad \textrm{ or } \quad \frac{M_{i+1} }{M_{i+1} \cap( M_i + L)} \cong \frac{M_{i+1}}{M_i}. \end{equation*}
      We claim that the latter case does not hold for all \(i\text{:}\) if it did, we would have then \(M_{i+1} \cap (M_i+L) = M_{i}\) for all \(i\text{,}\) so
      \begin{equation*} \frac{M_{i+1} + L}{M_{i+1}} \cong \frac{L+M_i}{(L+M_i) \cap M_{i+1}} =\frac{L+M_i}{M_i} \end{equation*}
      for all \(i\text{,}\) and hence \(L \cong (L+M_0)/M_0 \cong (L+M_n)/M_n \cong 0\text{,}\) contradicting that \(L\neq 0\text{.}\) Thus, for some \(i\text{,}\) the first case holds, and we can skip that \(i\) to obtain a composition series of length less than \(n\text{,}\) so \(\ell(\overline{M})<m\text{.}\)
    3. We proceed by induction on length again. Given a filtration of \(M\text{,}\) we can suppose that there is some nonzero proper submodule \(L\) in the filtration, since otherwise we could just take any composition series. Then \(L\) and \(\overline{M}\) has length less than \(M\text{.}\) The filtration up to \(L\) can be refined to a strict composition series by the induction hypothesis, and the filtration from \(L\) to \(M\) taken mod \(L\) can be refined to a strict composition series for \(\overline{M}\text{;}\) pulling back as in the remark above, we get the strict composition series we want.
    4. We show by induction on \(m\) that for any module of length \(m\text{,}\) all of its strict composition series are equivalent. Assume that \(\ell(M)=m\text{.}\) If \(m=1\text{,}\) the claim is clear since we are dealing with a simple module. Suppose that
      \begin{equation*} N_\bullet: \qquad 0 = N_0 \subsetneqq N_1 \subsetneqq \cdots \subsetneqq N_n = M \end{equation*}
      is another strict composition series for \(M\text{,}\) so \(n\geq m\text{.}\) If \(N_{n-1}=M_{m-1}\text{,}\) then since \(\ell(M_{m-1})\leq m-1\) the two composition series we have for \(M_{m-1}\) are equivalent by induction, so the two given series are equivalent.
      If \(N_{n-1} \neq M_{m-1}\text{,}\) since \(M/M_{m-1}\) is simple, \(M_{m-1}\) is not properly contained in \(N_{n-1}\text{,}\) so the image of \(M_{m-1}\) in \(M/N_{n-1}\) is nonzero, so equals all of \(M\text{,}\) which means that \(N_{n-1}+M_{m-1} = M\text{.}\) Set \(K=N_{n-1}\cap M_{m-1}\text{.}\) By the second isomorphism theorem, we then have
      \begin{equation*} \frac{M}{M_{m-1}} = \frac{M_{m-1}+N_{n-1}}{M_{m-1}} \cong \frac{N_{n-1}}{K} \end{equation*}
      and similarly \(M/N_{n-1} \cong M_{m-1} / K\text{,}\) and both of these modules are simple.
      Fix a strict composition series for \(K\text{:}\)
      \begin{equation*} K_{\bullet}: \qquad 0 = K_0 \subsetneqq K_1 \subsetneqq \cdots \subsetneqq K_k = K \end{equation*}
      and extend to a strict composition series for \(M_{m-1}\text{:}\)
      \begin{equation*} K'_{\bullet}: \qquad 0 = K_0 \subsetneqq K_1 \subsetneqq \cdots \subsetneqq K_k = K \subsetneqq M_{m-1}. \end{equation*}
      Since we also have the strict composition series
      \begin{equation*} M_{\bullet \leq m-1} : \qquad 0 = M_0 \subsetneqq M_1 \subsetneqq M_2 \subsetneqq \cdots \subsetneqq M_{m-1} \end{equation*}
      of length \(m-1\text{,}\) we must have that \(k=m-2\) and \(K'_{\bullet}\) is equivalent to \(M_{\bullet \leq n-1}\text{.}\) Thus, the composition factors of \(M_{\bullet \leq m-1}\) are those of \(K\) plus one copy of \(M_{m-1} / K \cong M/N_{n-1}\text{.}\)
      Now,
      \begin{equation*} K''_{\bullet}: \qquad 0 = K_0 \subsetneqq K_1 \subsetneqq \cdots \subsetneqq K_{m-2} = K \subsetneqq N_{n-1} \end{equation*}
      is a strict composition series for \(N_{n-1}\text{,}\) so \(n=m\text{.}\) Then, \(K''_{\bullet}\) is equivalent to the strict composition series
      \begin{equation*} N_{\bullet \leq n-1}: \qquad 0 = N_0 \subsetneqq N_1 \subsetneqq \cdots \subsetneqq N_{n-1}. \end{equation*}
      Thus, the composition factors of \(N_{\bullet \leq n-1}\) are those of \(K\) plus one copy of \(N_{n-1} / K \cong M/M_{n-1}\text{.}\)
      It follows that the composition series \(M_\bullet\) and \(N_\bullet\) are equivalent.

    Example 25.10.

    1. If \(K\) is a field, then a \(K\)-vector space of dimension \(n\) is a \(K\)-module of length \(n\text{.}\)
    2. If \(R\) is a \(K\)-algebra, and \(M\) is an \(R\)-module that as a \(K\)-vector space has dimension \(n\text{,}\) then \(\ell(M)\leq n\text{,}\) since the vector space dimension of a proper submodule is strictly smaller.
    3. The ring \(R=K[x]\) does not have finite length as a module over itself.
    4. \(\mathbb{Z}/p^n\) has length \(n\) as a \(\mathbb{Z}\)-module, with strict composition series
      \begin{equation*} 0 \subseteq \langle \overline{p^{n-1}} \rangle \subseteq \cdots \subseteq \langle \overline{p} \rangle \subseteq \mathbb{Z}/p^n. \end{equation*}
