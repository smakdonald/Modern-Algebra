    Chapter 1 Groups

    Section 1.1 First Definitions

    Subsection 1.1.1 Groups

    Definition 1.1. Binary Operation.
    A binary operation on a set \(S\) is a function
    \begin{equation*} -\cdot-:S\times S\to S,\text{given by } (x,y)\to x\cdot y. \end{equation*}
    Remark 1.2.
    We often write \(xy\) instead of \(x\cdot y\) for simplicity.
    Definition 1.3.
    A group is a pair \((G, \cdot)\) where \(G\) is a set and \(\cdot\) is a Binary Operation on \(G\text{,}\) called group multiplication, satisfying the following properties:
    1. (Associativity) for all \(x,y,z\in G\) we have \((x\cdot y)\cdot z=x\cdot (y\cdot z)\)
    2. (Identity Element) there exists \(e\in G\) such that \(e¬∑x=x¬∑e=x\) for all \(x\in G\)
    3. (Inverses) for each \(x\in G\text{,}\) there is an element \(y\in G\) such that \(xy=e=yx\text{.}\)
    Remark 1.4.
    Although a group is a pair, we will usually refer to the group by only naming the underlying set, \(G\text{.}\)
    Remark 1.5.
    We say that ‚Äú\(S\) is closed under the operation \(\cdot\)‚Äù, when we want to emphasize that for any \(x,y\in S\) the result of the operation, \(xy\text{,}\) is an element of \(S\text{.}\) However note that closure is really part of the definition of a binary operation on a set, and it is implicitly assumed whenever we consider such an operation.
    Definition 1.6.
    A group \(G\) is an abelian group if \(\cdot\) is commutative; i.e., \(x \cdot y = y \cdot x\) for all \(x,y \in G\text{.}\) Often, but not always, the group operation for an abelian group is written as \(+\) instead of \(\cdot\text{.}\) In this case the inverse of an element \(x\) is written \(-x\text{.}\)
    Proposition 1.7.
    In a group \(G\text{,}\) the element \(e\) satisfying the second axiom of Definition¬†1.3 is unique, and we thus refer to it as the identity element of \(G\text{.}\)
    Proof.
    If \(ex = x = xe\) and \(e'x = x = xe'\) for all \(x\text{,}\) then \(e = ee' = e'\text{.}\)
    Proposition 1.8.
    In a group \(G\text{,}\) for each \(x\text{,}\) the element \(y\) satisfying the last axiom of Definition¬†1.3 is unique.
    Proof.
    Proof. For a given \(x\text{,}\) if \(yx = xy = e\) and \(zx = xz = e\) fo some \(y\) and \(z\text{,}\) then \(z = ez = (yx)z = y(xz) = ye = y\text{.}\)
    Remark 1.9.
    We will call the element \(y\) satisfying the last axiom of Definition¬†1.3 the inverse of \(x\) and we will henceforth denote it by \(x^{-1}\text{.}\)
    Lemma 1.10. Properties of Groups.
    If \(G\) is a group and \(x,y,z, a_1,\ldots,a_n \in G\text{,}\) then:
    1. if \(xy = xz\text{,}\) then \(y=z\text{.}\)
    2. if \(yx = zx\text{,}\) then \(y = z\text{.}\)
    3. \((x^{-1})^{-1} = x\text{.}\)
    4. \((a_1 \dots a_n)^{-1} = a_n^{-1} \dots a_1^{-1}\text{.}\)
    5. \((x^{-1}yx)^n = x^{-1}y^nx\text{.}\)
    Proof.

    Subsection 1.1.3 Dihedral Groups

    We now discuss the important example of dihedral groups in detail.
    Definition 1.15.
    For any integer \(n \geq 3\text{,}\) let \(P_n\) denote a regular \(n\)-gon. An isometry of the plane is a function \(f: {\mathbb{R}}^2 \to {\mathbb{R}}^2\) that is a bijection and preserves the Euclideal distance \((d(f(A),f(B))=d(A,B)\) for any \(A,B\in {\mathbb{R}}^2\)). A symmetry of \(P_n\) is an isometry that maps \(P_n\) to itself.
    Remark 1.16.
    By the latter I don't mean that \(f\) fixes each of the points of \(P_n\text{,}\) but rather that \(f(P_n) = P_n\text{,}\) that is every point of \(P_n\) is mapped to a (possibly different) point of \(P_n\) and every point of \(P_n\) is the image of some point in \(P_n\) via \(f\text{.}\) It is clear that the composition of two symmetries of \(P_n\) is again a symmetry of \(P_n\text{,}\) so that composition is a binary operation on \(D_n\text{.}\)
    Definition 1.17.
    The dihedral group \(D_{2n}\) is the set of symmetries of the regular \(n\)-gon \(P_n\) equipped with the binary operation given by composition.
    Problem 1.18.
    Prove that the dihedral group \(D_{2n}\) is indeed a group.

    Subsection 1.1.4 Symmetric Groups

    Let's introduce another very important example: symmetric groups.
    Definition 1.19.
    A permutation of a set \(X\) is a bijective function \(\sigma:X\to X\text{.}\)
    Definition 1.20.
    For any \(n\in\N\text{,}\) the symmetric group is the set \(S_n\) of all permutations of the set \(\{1,\dots,n\}\) equipped with the composition of functions as its binary operation.
    Remark 1.21.
    For any finite set \(|X|=n\text{,}\) we may think of \(S_n\) as the permutation group of \(X\text{,}\) which is sometimes written \(\operatorname{Perm}(X)\text{.}\)
    Problem 1.22.
    Prove that \(S_n\) is indeed a group for all \(n\in\N\text{.}\)
    We can write an element \(\sigma\) of \(S_n\) as a table of values:
    \begin{equation*} \begin{pmatrix} 1 & 2 & 3 & \cdots & n \\ \sigma(1) & \sigma(2) & \sigma(3) & \cdots & \sigma(n) \end{pmatrix} \end{equation*}
    Definition 1.23.
    If \(i_1, \dots, i_m\) are distinct integers between \(1\) and \(n\text{,}\) then \(\sigma=(i_1 \, i_2 \, \dots i_m)\) denotes the element of \(S_n\) that satisfies
    \begin{equation*} \sigma(i_1)=i_2, \sigma(i_2)=i_3, \dots, \sigma(i_{m-1})=i_m, \sigma(i_m)=i_1 \text{ and } \sigma(k)=k \text{ for } k\in[n] \setminus \{i_1, \dots, i_m\}. \end{equation*}
    Such a permutation is called a cycle or an \(m\)-cycle if we want to emphasize its length. A \(2\)-cycle is often called a transposition.
    Problem 1.24.
    Determine the number of \(1\)-cycles in \(S_n\text{.}\)
    Remark 1.25.
    Note that distinct lists of integers represent the same cycle if they are cyclical rearrangements of each other, e.g., \((1 \, 2 \, 3) = (2 \, 3 \, 1)\text{.}\) However, \((1 \, 2 \, 3) \neq (2 \, 1 \, 3)\text{.}\)
    Problem 1.26.
    Disjoint cycles commute, that is, if \(i_1 , i_2 \, \dots i_m, j_1 , j_2 \, \dots j_k\in [n]\text{,}\)
    \begin{equation*} \sigma_1=(i_1 \, i_2 \, \dots i_m), \quad \sigma_2=(j_1 \, j_2 \, \dots j_k) \end{equation*}
    and
    \begin{equation*} \{i_1 , i_2 , \dots i_m\}\cap \{j_1 , j_2 , \dots j_k\}=\emptyset \end{equation*}
    then \(\sigma_1 \circ \sigma_2=\sigma_2\circ \sigma_1\text{.}\)
    Proposition 1.27.
    1. Each \(\sigma\in S_n\) can be written as a product (composition) of disjoint cycles, and such a factorization is unique up to the ordering of the factors.
    2. Each \(\sigma\in S_n\) can be written a product of transpositions; i.e., \(S_n\) is generated by transpositions.
    Remark 1.28.
    For the uniqueness part of statement (1) in the above proposition one needs to establish a convention regarding 1-cycles, that is one needs to stipulate either that the 1-cycles will not be recorded (which gives the shortest such factorization) or that all the 1-cycles will be recorded (which gives the longest such factorization, but also the only one that makes it clear what the number \(n\) is).

    Subsection 1.1.5 The Quaternions

    For our last example we mention the group of quaternions, written \(Q_8\text{.}\)
    Definition 1.29.
    The quaternion group \(Q_8\) is a group with \(8\) elements
    \begin{equation*} Q_8=\{ 1, -1, i, -i, j, -j, k, -k \} \end{equation*}
    satisfying the following relations: \(1\) is the identity element and
    \begin{equation*} i^2 = -1, j^2 = -1, k^2 =-1, ij = k, jk = i, ki = j, \end{equation*}
    \begin{equation*} (-1)i = -i, (-1)j = -j, (-1)k = -k, (-1)(-1) = 1. \end{equation*}
    Problem 1.30.
    Verify that \(Q_8\) is a group. Provide a specific example to show that \(Q_8\) is not an abelian group.

    Subsection 1.1.6 Order

    Definition 1.31.
    In a group \(G\text{,}\) the order of an element \(x\) is the least positive integer \(n\) such that \(x^n = e\text{.}\) If no such \(n\) exists, we say \(x\) has infinite order. We write \(|x|\) for the order of \(x\text{.}\)
    Definition 1.32.
    The order of a group \(G\) is the cardinality of the set \(G\text{,}\) denoted \(|G|\text{.}\)
    Problem 1.33.
    Prove that any group of order \(4\) is abelian.
    Solution.
    Proposition 1.34.
    Let \(G\) be a group and \(x \in G\) any element.
    1. Prove that if \(|x| = n\text{,}\) then \(e, x, \dots, x^{n-1}\) are all distinct elements of \(G\text{.}\)
    2. Prove that if \(|x| = \infty\text{,}\) then \(x^i \ne x^j\) for all integers (positive or negative) \(,j\) with \(i \ne j\text{.}\)
    3. Conclude \(|x| \leq |G|\) in all cases.
    Problem 1.35.
    Prove \(|S_n|=n!\text{.}\)
    Problem 1.36.
    Prove that the order of a permutation is the least common multiple of the lengths of the cycles it is a product of.
    Problem 1.37.
    Let \(G, H\) be groups and let \(x, y\) be elements of \(G\text{.}\)
    1. Prove that \(|x|=\left|\{x^n \ | n\in \Z \}\right|\) and in particular \(|x| \leq |G|\text{.}\)
    2. Prove or disprove: if \(x,y\) have finite order then \(xy\) has finite order.
    Problem 1.38.
    Let \(G\) be a group.
    1. Let \(g \in G\) be an element of finite order. Show that \(g^m\) has finite order for any integer \(m \geq 0\text{,}\) and in fact
      \begin{equation*} |g^m| = \frac{\lcm(m,|g|)}{m} = \frac{|g|}{\gcd(m, |g|))}. \end{equation*}
    2. Prove that for all \(g, h\) in \(G\text{,}\) \(|gh| = |hg|\) holds.
    Proposition 1.39.
    1. Every element in \(D_{2n}\) can be written as \(r^j\) or \(r^js\) for \(0 \leq j \leq n-1\text{.}\)
    2. \(\displaystyle |D_{2n}|=2n\)
    Lemma 1.40.
    If \(x^m = e\) then \(|x| \mid m\text{.}\)

    Section 1.2 Generators and Cyclic Groups

    Definition 1.41.

    A subset \(S\) of a group \(G\) is said to generate \(G\) if every element of \(G\) is equal to a product of elements of \(S\) and their inverses (with repetitions allowed).
    Assume that the regular \(n\)-gon \(P_n\) is drawn in the plane with its center at the origin and one vertex on the \(x\) axis. If \(r\) denotes rotation about the origin by \(\frac{2\pi}{n}\) radians counter-clockwise, then \(r \in D_{2n}\text{.}\) It‚Äôs inverse is rotation by \(2 \pi/n\) clock-wise. For another example, for any line of symmetry of \(P_n\text{,}\) reflection about that line gives an element of \(D_{2n}\text{.}\) By our convention for how to draw \(P_n\text{,}\) the \(x\)-axis is a line of symmetry for \(P_n\text{,}\) and we let \(s\) denote reflection about the \(x\)-axis.

    Problem 1.42.

    1. Prove \(|r|=n\) and \(|s|=2\text{.}\)
    2. Prove that \(sr\) is a reflection and hence that \(srsr=1_{{\mathbb{R}}^2}\text{.}\)

    Problem 1.43.

    1. Prove that \(D_{2n}\) is generated by \(r\) and any reflection.
    2. Prove that if \(p\ge 3\) is prime then \(D_{2p}\) is generated by any two distinct reflections.

    Problem 1.44.

    For \(n\ge 2\) prove that \(S_n\) is generated by \((12)\) and the \(n\)-cycle \((12\cdots n)\text{.}\)

    Subsection 1.2.1 Cyclic Groups

    Definition 1.45.
    If \(G\) is a generated by a single element, i.e. \(G = \langle x \rangle\) for some \(x \in G\text{,}\) then \(G\) is called a cyclic group.
    Problem 1.46.
    Let \(G=\langle x\rangle\) be a cyclic group. Then \(G=\langle x^{-1}\rangle\text{.}\)
    Problem 1.47.
    1. Prove \((\mathbb{Z},+)\) is generated by \(1\) or \(-1\)
    2. Prove \((\mathbb{Z}/(n),+)\) is generated by \([a]_n\) for any integer \(a\) such that \(\operatorname{gcd}(a,n)=1\text{.}\)
    Problem 1.48.
    Prove that \((\mathbb Q, +)\) is not a cyclic group.
    Proposition 1.49.
    Prove that any cyclic group is abelian.
    Problem 1.50.
    Prove that \(\operatorname{GL}_2(\mathbb Z/(2))\) is not cyclic.

    Section 1.3 Subgroups

    Subsection 1.3.1 Definition and Examples

    Definition 1.51.
    A nonempty subset \(H\) of a group \(G\) is called a subgroup provided \(H\) is a group under the multiplication law of \(G\text{.}\) The fact that \(H\) is a subgroup of \(G\) is denoted \(H\leq G\) or \(H<G\) if we also mean that \(H\neq G\text{.}\)
    Lemma 1.52. Subgroup Tests.
    1. If a subset \(H\) of a group \(G\) is nonempty and closed under multiplication and inversion, then \(H\) is a subgroup.
    2. If a subset \(H\) of a group \(G\) is nonempty and satisfies for all \(x,y \in H\text{,}\) \(xy^{-1} \in H\text{,}\) then \(H\) is a subgroup.
    Example 1.53.
    Prove the following:
    1. \(\{e_G\}\) and \(G\) are the trivial subgroups of \(G\text{.}\)
    2. \(\displaystyle \Z\leq\Q\leq\R\leq\C\)
    3. \(\displaystyle \Z^\times\leq\Q^\times\leq\R^\times\leq\C^\times\)
    4. Let \(n\in\N\) and define \(n\Z=\{x\in\Z|x=nk\;\text{for some}\;k\in\Z\}\text{.}\) Prove \(n\Z\leq\Z\) for all \(n\text{.}\)
    5. If \(H\) is a subgroup of \(G\) and \(K\) is a subgroup of \(H\text{,}\) then \(K\) is a subgroup of \(G\text{.}\)
    6. If \(H_\alpha\) is a subgroup of \(G\) for all \(\alpha\) in an index set \(J\text{,}\) then \(H=\bigcap_{\alpha\in J} H_\alpha\) is a subgroup of \(G\text{.}\)
    7. Let \(G\) be a group. For any \(g\in G\text{,}\) prove that the set \(Z_G(g)=\{x\in G|gx=xg\}\) is a subgroup of \(G\text{.}\)
    Problem 1.54.
    Let \(G\) be a group and \(H\text{,}\) \(H'\) subgroups. Prove that \(H\cup H'\) is a subgroup if and only if \(H\subseteq H'\) or \(H'\subseteq H\text{.}\)
    Problem 1.55.
    Let \(H\) be a [[Mathematics/Definitions/Subgroup|subgroup]] of \(G\) and fix some element \(g\in G\text{.}\) Prove that \(gHg^{-1}=\{ghg^{-1} \mid h\in H\}\) is a subgroup of \(G\) of the same [[Mathematics/Definitions/Order|order]] as \(H\text{.}\)
    Definition 1.56.
    The alternating group \(A_n\) is the subset of all even cycles of \(S_n\text{.}\)
    Problem 1.57.
    Prove that \(A_n\) is a subgroup of \(S_n\text{.}\)
    Definition 1.58.
    For any field \(F\text{,}\) the special linear group
    \begin{equation*} \SL_n(F)=\{A \mid A = n\times n \;\text{matrix with entries in}\; F, \det(A)=1_F\} \end{equation*}
    Problem 1.59.
    Prove that \(SL_n(F)\) is a subgroup of \(GL_n(F)\text{.}\)

    Subsection 1.3.2 The Center of a Group

    Definition 1.60.
    The center of a group \(G\) often written \(Z(G)\text{,}\) is the set of elements of \(G\) that commute with every element of \(G\text{.}\) That is,
    \begin{equation*} Z(G)=\{x \in G \, | \, xy = yx\,\text{ for all $y \in G$}\}. \end{equation*}
    Problem 1.61.
    Prove that \(Z(G)\) is an abelian subgroup for any group \(G\text{.}\)
    Problem 1.62.
    Find \(Z(D_{2n})\) for \(n \geq 3\text{.}\) Your answer will depend on whether \(n\) is even or odd.

    Subsection 1.3.3 Generated Subgroups

    Definition 1.63.
    Given a group \(G\) and a subset \(X\) of \(G\text{,}\) the subgroup of \(G\) generated by \(X\) is
    \begin{equation*} \langle X \rangle := \bigcap_{H \leq G, H \supseteq X} H. \end{equation*}
    If \(X=\{x\}\) is a set with one element then we write \(\langle X \rangle=\langle x \rangle\) and we refer to this as the cyclic subgroup generated by \(x\text{.}\)
    Lemma 1.64.
    For a subset \(X\) of \(G\text{,}\) the elements of \(\langle X \rangle\) can be described as:
    \begin{equation*} \langle X \rangle = \left\{x_1^{j_1} \cdots x_m^{j_m} \mid m \geq 0, j_1, \dots, j_m \in \mathbb{Z}\text{ and }x_1, \dots, x_m \in X \right\}. \end{equation*}
    Proof.
    Let \(S= \left\{x_1^{j_1} \cdots x_m^{j_m} \mid m \geq 0, j_1, \dots, j_m \in \mathbb{Z}\text{ and }x_1, \dots, x_m \in X \right\}\text{.}\) Since \(\langle X \rangle\) is a subgroup that contains \(X\text{,}\) it is closed under products and inverses, and thus must contain all elements of \(S\text{.}\)
    For the opposite containment, we just need to show the set \(S\) really is a subgroup. We use the one step test:
    • \(S \ne \emptyset\) since we allow \(m = 0\) and declare the empty product to be \(e_G\text{.}\)
    • If \(x_1^{j_1} \cdots x_m^{j_m}\) and \(y_1^{i_1} \cdots y_n^{i_m}\) are in \(S\) then
      \begin{equation*} x_1^{j_1} \cdots x_m^{j_m}(y_1^{i_1} \cdots y_n^{i_m})^{-1}=x_1^{j_1} \cdots x_m^{j_m} y_n^{-i_m} \cdots y_1^{-i_1} \end{equation*}
      is also in \(S\text{.}\)
    Therefore \(S\leq G\) and \(X\subseteq S\) (by taking \(m=1\) and \(j_1=1\)) and by the minimality of \(\langle X \rangle\) we obtain \(\langle X \rangle\subseteq S\text{.}\)
    Problem 1.65.
    The Lemma implies that for an element \(x\) of a group \(G\text{,}\) \(\langle x\rangle=\{x^j \mid j\in \mathbb{Z}\}\text{.}\)
    Problem 1.66.
    Verify \(D_{2n}=\langle r,s \rangle\text{,}\) meaning that \(D_{2n}\) is the subgroup of \(D_{2n}\) generated by \(\{r,s\}\text{.}\)
    Problem 1.67.
    Prove that \(A_n\) is the subgroup of \(S_n\) generated by all produts of \(2\)-cyclies (disjoint or not). In symbols,
    \begin{equation*} A_n=\langle (i \ j )(k \ l) \mid i,j,k,l \in \{1, \ldots, n\}\rangle. \end{equation*}
    Problem 1.68.
    Consider the set \(\mu=\{z\in \C \setminus \{0\} \mid z^n = 1 \text{ for some integer }n \geq 1\}\text{,}\) which is a subgroup of \(( \C \setminus \{0\} , \cdot)\) (a fact you need not prove).
    1. Prove that for each integer \(m \geq 1\text{,}\) there is a unique subgroup \(H_m\leq \mu\) with \(|H_m|=m\) and that \(H_m\) is cyclic.
    2. Prove that every finitely generated subgroup of \(\mu\) has finitely many elements.
    3. Prove that \(\mu\) is not finitely generated.
