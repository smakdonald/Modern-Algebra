    Definition 1.1. Binary Operation.

    A binary operation on a set \(S\) is a function
    \begin{equation*} -\cdot-:S\times S\to S,\text{given by } (x,y)\to x\cdot y. \end{equation*}

    Remark 1.2.

    We often write \(xy\) instead of \(x\cdot y\) for simplicity.

    Definition 1.3. Group.

    A group is a pair \((G, \cdot)\) where \(G\) is a set and \(\cdot\) is a Binary Operation on \(G\text{,}\) called group multiplication, satisfying the following properties:
    1. (Associativity) for all \(x,y,z\in G\) we have \((x\cdot y)\cdot z=x\cdot (y\cdot z)\)
    2. (Identity Element) there exists \(e\in G\) such that \(e¬∑x=x¬∑e=x\) for all \(x\in G\)
    3. (Inverses) for each \(x\in G\text{,}\) there is an element \(y\in G\) such that \(xy=e=yx\text{.}\)

    Remark 1.4.

    Although a group is a pair, we will usually refer to the group by only naming the underlying set, \(G\text{.}\)

    Remark 1.5.

    We say that ‚Äú\(S\) is closed under the operation \(\cdot\)‚Äù, when we want to emphasize that for any \(x,y\in S\) the result of the operation, \(xy\text{,}\) is an element of \(S\text{.}\) However note that closure is really part of the definition of a binary operation on a set, and it is implicitly assumed whenever we consider such an operation.

    Theorem 1.6. Properties of Groups.

    1. (Unique Identity) In a group \(G\text{,}\) the element \(e\) satisfying the second axiom of Definition¬†1.3 is unique, and we thus refer to it as the identity element of \(G\text{.}\)
    2. (Unique Inverses) In a group \(G\text{,}\) for each \(x\text{,}\) the element \(y\) satisfying the last axiom of Definition¬†1.3 is unique, and we thus refer to it as the the inverse of \(x\) and we will henceforth denote it by \(x^{-1}\text{.}\)
    3. (Cancellation) if \(xy = xz\text{,}\) then \(y=z\text{.}\) Similarly, if \(yx = zx\text{,}\) then \(y = z\text{.}\)

    Proof.

    1. Suppose there exist two elements \(e\) and \(e'\) such that \(ex = x = xe\) and \(e'x = x = xe'\) for all \(x\text{.}\) Then
      \begin{equation*} e = ee' = e'. \end{equation*}
      Thus only one element can satisfy this property, making the identity element unique.
    2. For a given \(x\text{,}\) if \(yx = xy = e\) and \(zx = xz = e\) fo some \(y\) and \(z\text{,}\) then \(z = ez = (yx)z = y(xz) = ye = y\text{.}\)

    Definition 1.7. Abelian Group.

    A group \(G\) is an abelian group if \(\cdot\) is commutative; i.e., \(x \cdot y = y \cdot x\) for all \(x,y \in G\text{.}\) Often, but not always, the group operation for an abelian group is written as \(+\) instead of \(\cdot\text{.}\) In this case the inverse of an element \(x\) is written \(-x\text{.}\)
    Abelian groups are named after the Norwegian mathematician Niels Henrik Abel, who made significant contributions to the study of algebra and number theory in the early 19th century.
    Abel's work on abelian groups helped to establish the importance of commutativity in algebraic structures, and his ideas laid the foundation for the development of the theory of commutative rings and fields, which will be studied in much greater depth later.

    Example 1.8. Group Examples.

    1. The trivial subgroup is the group with a single element \(\{e\}\)
    2. \((\Z,+), (\Q,+), (\R,+),\) and \((\C,+)\) are all abelian groups, where \(+\) denotes traditional addition.
    3. \(\Q^+\) and \(\R^+\) are groups under traditional multiplication.
    4. For any positive integer \(n\text{,}\) let
      \begin{equation*} \operatorname{GL}_n(\C)=\{ \text{invertible } n \times n \;\text{matrices with entries in}\; \C\}. \end{equation*}
      By invertible I mean those matrices that have two-sided inverses, but in turns out that if an \(n \times n\) matrix has a left inverse then it it is automatically a right inverse too, and vice versa. Then \(\operatorname{GL}_n(\C)\) is a non-abelian group under matrix multiplication.
    5. For any \(n\text{,}\) let \(\mathbb{Z}/(n)\) denote the integers modulo \(n\text{.}\) Then \((\mathbb{Z}/(n), +)\) is an abelian group where \(+\) denotes addition modulo \(n\text{.}\)

    Definition 1.11. Order.

    1. In a group \(G\text{,}\) the order of an element \(x\) is the least positive integer \(n\) such that \(x^n = e\text{.}\) If no such \(n\) exists, we say \(x\) has infinite order. We write \(|x|\) for the order of \(x\text{.}\)
    2. The order of a group \(G\) is the cardinality of the set \(G\text{,}\) denoted \(|G|\text{.}\)

    Example 1.12.

    1. \(\displaystyle |e|=1\)
    2. \(|1|=n\) in \(\Z/(n)\)
    3. In the additive groups \(\Z\text{,}\) \(\Q\text{,}\) \(\R\text{,}\) and \(\C\) every nonzero (i.e., nonidentity) element has infinite order.

    Theorem 1.13. Properties of Order.

    1. An element \(g\) of a group \(G\) has order \(1\) if and only if \(g=e\text{.}\)
    2. \(\displaystyle |g|=|g\inv|\)
    3. If \(g^p=e\) for \(g\neq e\) and \(p\) prime, then \(|g|=p\text{.}\)

    Proposition 1.14. Marley's Conjecture.

    Every group of order \(4\) is abelian.

    Proof.

    Let \(G\) be a group with \(4\) elements, and suppose there are elements \(a,b\in G\) such that \(ab\neq ba\) for some elements \(a, b\in G\text{.}\) Since \(ab \neq ba\text{,}\) we must have \(a \neq b\text{,}\) \(a \neq e\text{,}\) and \(b\neq e\text{.}\) Since \(G\) has only \(4\) elements and \(ab \neq ba\text{,}\) either \(ab \in \{e, a, b\}\) or \(ba \in\{e, a, b\}\text{.}\) Without loss, say the former occurs. But \(ab = e\) implies \(b=a\inv\) and we know \(a\inv\) commutes with \(a\text{,}\) and hence this is not possible. If \(ab = a\text{,}\) then \(b = e\) and if \(ab = b\) then \(a = 1\text{,}\) both of which are impossible. Since \(a, b\) were arbitrary, \(G\) must be abelian.

    Lemma 1.15.

    Let \(G\) be a group. Suppose \(g^k=g^m\) and \(k<m\text{.}\)
    1. then \(g^{m-k}=e\)
    2. then \(|g|\leq m-k\)

    Theorem 1.16.

    Let \(G, H\) be groups and let \(x, y\) be elements of \(G\text{.}\) Then \(|x|=\left|\{x^n \ | n\in \Z \}\right|\) and in particular \(|x| \leq |G|\text{.}\)

    Lemma 1.17.

    If \(x^m = e\) then \(|x| \mid m\text{.}\)

    Let \(g_1\times\cdots\times g_n\in G_1\times\cdots\times G_n\text{.}\) Then \(|g_1\times\cdots\times g_n|=\lcm(|g_1|,\dots,|g_n|\text{.}\)
