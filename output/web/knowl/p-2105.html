Let \(\cA\) be a set. The set \(\cA\) is called a family of sets if all the elements of A are sets. The family of sets \(\cA\) is indexed by \(I\text{,}\) denoted \(\cA = \{A_i\}_{i\in I}\text{,}\) if there is a non-empty set \(I\) such that there is an element \(A_i\in \cA\) for each \(i\in I\text{,}\) and that every element of \(\cA\) equals \(A_i\) for exactly one \(i\in I\text{.}\)
