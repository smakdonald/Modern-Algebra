2. Group Homomorphisms.
Prove that each of the following are group homomorphisms
  1. The function \(f: \Z\to \Z\) defined by \(f(n) = 2n\text{.}\)
  2. The function \(f: \R \to \R\) defined by \(f(x) = x^2\text{.}\)
  3. The function \(f: \Z/(6)\to\Z/(3)\) defined by \(f([n]_6) = [n]_3\)
3. Finding Injective Homomorphisms.
  1. Find, with justification, an injective group homomorphism \(f:S_n \to \GL_n(\R)\text{.}\)
  2. Find, with justification, an injective group homomorphism \(g:D_{2n}\to S_n\text{.}\)
