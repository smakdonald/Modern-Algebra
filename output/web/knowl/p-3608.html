Let X be a set.
  1. The following are equivalent:
    1. X is finite.
    2. There is an onto function \{1,...,n\} ↠ X for some n \in \N.
    3. There is a one-to-one function X \hookrightarrow \{1,...,n\} for some n \in \N.
  2. The following are equivalent:
    1. X is infinite.
    2. There is an onto function X ↠ \N .
    3. There is a one-to-one function \N \hookrightarrow X.
  3. The following are equivalent:
    1. X is countable.
    2. There is an onto function \N \to X.
    3. There is an injection X \hookrightarrow \N .
